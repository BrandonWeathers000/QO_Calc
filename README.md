# QO_Calc
This is a calculator based on the queue data structure. I hope you find it useful.

* Please refer to the pdf documentation to use this app.*

## How to run
1. Download JavaFX
2. javac --module-path ~/javafx-sdk-25.0.3/lib --add-modules javafx.controls *.java
3. java --module-path ~/javafx-sdk-25.0.3/lib --add-modules javafx.controls MainScreen

## Development Roadmap
1. Migration to the BigDecimal class
2. Addition useful functions
    1. Log (base 10, 2, and e)
    2. Base n roots 
    3. Arithmetic mean, geometric mean, median, mode, and standard deviation
    4. Factorials, combinatorics, and permutations
    5. Trig functions (sine, cosine, tangent, arcsine, arccosine, arctangent, hyperbolic sine, hyperbolic cosine, and hyperbolic tangent)
3. Support for four queues
4. Constants (pi, e, and phi)
5. Special exponentiation (e^x, 10^x)
